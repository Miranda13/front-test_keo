# front-test_keo
Angular front to keo test

Clone repository from git

```bash
# clone the project
git clone https://github.com/Miranda13/front-test_keo.git
# enter the project directory
cd front-test_keo
```
Run local server
```bash
# install dependency
npx npm install
or
npm install
# start server
npx ng serve
# Whenever you want to change the port just run
npx ng dev --port = 9528
```
